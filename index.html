About This Tool

This tool uses a machine learning model to predict the likelihood of breast cancer recurrence based on tumor characteristics. It is built as an educational project and should not be used for actual medical decisions.

  • Dataset: Based on the Breast Cancer Wisconsin dataset (sklearn)
  • Algorithms: Logistic Regression & Random Forest (best model selected)
  • Input Features: Age, Tumor Size, Lymph Nodes, Grade, ER/PR, HER2
